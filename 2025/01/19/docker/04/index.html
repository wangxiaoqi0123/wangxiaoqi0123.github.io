Dockerfile

发表于 2025-01-19 更新于 2025-01-23 分类于 docker

Dockerfile 用于自动化构建 Docker 镜像，定义镜像的环境、依赖和行为。
参考: https://docs.docker.com/reference/dockerfile/

示例

前端vue，后端nodejs的websocket服务，打包成docker镜像示例

前端镜像打包

1.本地vue项目构建打包出dist文件
2.编写nginx/nginx.conf文件
3.编写Dockerfile

4.将dist、nginx.conf、Dockerfile推送到服务器 /app/websocket/client 目录下
5.docker build -t websocketclient:v1.0 . (构建镜像，注意最后的点号表示当前目录为构建上下文)

后端镜像打包

1.编写Dockerfile

2.将Dockerfile、index.js工程文件推送到服务器 /app/websocket/server 目录下
3.docker build -t websocketserver:v1.0 . (构建镜像，注意最后的点号表示当前目录为构建上下文)

命令

# docker
docker compose
