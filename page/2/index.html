Vue 父子组件通讯

发表于 2020-06-19 更新于 2020-07-10

子组件向父组件传递消息

  1. 使用 ref 的方式
    (1) 父组件下 子组件标签 上标识 ref=”” 标识
    (2) 父组件 通过 this.$refs.标识 === 子组件Vue实例

  2. 父组件通过 $children 的方式获取子组件消息
    (1) 父组件 通过 this.$children === [子组件Vue实例 ,…] 子组件Vue实例的数组

  3. 子组件使用 $emit 的方法 发布订阅向父组件 传递消息
    (1) 子组件 发布 this.$emit(“订阅事件” , “要传递给父元素的数据”)
    (2) 父组件下 子组件标签 上添加监听事件 @订阅事件 = callback
    (3) 要接受的参数 在callback 中实现传递

父组件向子组件传递消息

  1. 使用 props 方式
    (1) 父组件下 子组件标签 通过数据属性向子组件传递数据 :数据名=”数据”
    (2) 子组件 同过 props:[“数据名”] 的方式接受数据
  2. 使用 $parent 的方式获取父元素上的信息
    (1) 子组件 通过 this.$parent == 父组件Vue实例 (父组件唯一所以不是数组)

兄弟组件之间传递消息(非父子组件)

  1. 只有通过 Vue实例 实现通讯的一种方式
    (1) 创建一个 Vue实例 专门用于实现兄弟元素之间的通讯 var bus = new Vue()
    (2) 发送消息的兄弟 在 methods 中 使用 bus.$emit(“名称” , 数据)
    (3) 接受消息的兄弟 在 mounted (生命周期回调函数) 使用 bus.$on(“名称”,callback) 回调函数中接受消息

  2. router-link 组件名称

  3. this.$router.push(‘路由路径’)

  4. this.$router.push({name:’组件名称’})

页面的信息对象
this.$route

Vue 组件中跳转 带参数

  1. 跳转传参
  • 方式一 使用params 传递参数
    this.$router.push({path:’组件路径/‘,params:{变量名:值}})
    this.$router.push({name:’组件名称’,params:{变量名:值}})
  • 方式二 使用query 传递参数
    this.$router.push({path:’组件路径/‘,query:{变量名:值}})
    this.$router.push({name:’组件名称’,query:{变量名:值}})
  1. 接受参数
  • this.$route.params.变量名 (一般在 mounted 中接受参数) 页面路由看不见 页面刷新参数消失
  • this.$route.query.变量名 (一般在 mounted 中接受参数) 页面路由显示 页面刷新参数不消失

Vue 路由守卫

发表于 2020-06-19 更新于 2020-07-10

beforeRouteEnter 进入组件时路由守卫

beforeRouteEnter:(to,from,next)=>{}

to 进入到哪个路由(默认当前)
from 从哪个路由进入
next 路由拦截器 第一个参数 等价于this

beforeRouteLeave 离开组件时路由守卫

beforeRouteLeave:(to,from,next)=>{}

from 从哪个路由离开 (默认当前)
to 离开到哪个路由
next 路由拦截器 第一个参数 等价于this
