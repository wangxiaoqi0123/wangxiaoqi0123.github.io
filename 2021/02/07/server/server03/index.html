洋葱圈模型

  • 洋葱圈模型图

    当 koa2 server 接受到一个请求时，会先经过第一个洋葱圈（如图外层最大的洋葱圈），依次再进入内层洋葱圈，最内层的洋葱圈最后到达但会最先被执行完成，第一个洋葱圈最先到达但会最后执行完成。
