Node操作mongodb

发表于 2021-02-23 更新于 2023-10-20 分类于 node

mongodb模块

  • 安装
  • mongodb模块的文档
    • https://www.npmjs.com/package/mongodb
  • node操作mongodb 连接mongodb，切换到数据库，找到对应集合，进行增删改查。具体参考mongodb模块的文档 https://www.npmjs.com/package/mongodb

mongoose模块

  • 安装
  • mongoose模块文档
    • https://www.npmjs.com/package/mongoose （npm）
    • https://mongoosejs.com/ （官方文档）

  • mongoose连接数据库 mongoose.connect(‘mongodb://username:password@host:port/database’); 如果数据库开启校验 需要账户密码时 使用此规则对数据库进行连接。

  • Schema定义数据格式的规范，以Model规范Collection Schame文档 https://mongoosejs.com/docs/api/schema.html

  • Model 对文档进行增删改查 model 操作集合 即可以通过回调函数的形式读取操作结果，也可以通过Promise.then读取结果
    model 的api 文档具体请查看 https://mongoosejs.com/docs/api/model.html
