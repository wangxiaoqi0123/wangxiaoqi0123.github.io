细节使用

方式一 state + mutations

  1. 初始化变量

    1. state 对象中 初始化变量
    2. mutations 对象中 定义 一个函数用于 给变量赋值

  2. 传入参数 给变量赋值 使用 this.$store.commit()

  3. 在其他位置读取 store 中的变量

    1. 在需要引用的变量中引入 mapState 方法 import { mapState } from “vuex”;
    2. 直接在 计算属性 computed 中映射使用

方式二 state + mutations + actions

  1. state + mutations + actions

    1. state 对象中 初始化变量
    2. mutations 对象中 定义 一个函数用于 给变量赋值
    3. 用法一 actions 对象中 定义一个函数用于 提交 mutations中的函数 用法二 actions 对象中 可以使用异步的函数

  2. 传入参数 给变量赋值 使用 this.$store.dispatch()

    • 方式二 使用 this.$store.dispatch() 传参 执行异步函数
