NodeJS是什么

  • 是运行在服务端的JavaScript
  • 都是JavaScript，但又有所不同
  • v8引擎，事件驱动，非阻塞I/O

为什么学它

  • 快速上手一门服务端语言
  • 了解前后端的交互流程
  • 防止后端甩锅

模块

  • CommonJS规范
  • 自定义模块
    • 创建（一个js文件就是一个模块）
    • 导出 (module.exports=xxx)
    • 引用（require，js后缀名可省）
  • 内置模块
    • fs
      • 操作目录（mkdir，readdir，rename，rmdir）
      • 操作文件（writeFile，appendFile，readFile，unlink，stat）
    • path
      • 获取：__dirname，dirname，basename，extname
      • 组合：join，resovle
    • http
      • 创建服务，接收请求，解析参数，返回数据
      • 静态资源服务
        • 获取pathname
        • 使用readFile
        • 设置content-type
    • url
      • parse
      • format
    • querystring
      • parse
      • stringify
      • escape
      • unescape
  • 第三方模块
    • 定位到项目根目录
    • 初始化项目
    • 安装模块
      • npm
      • cnpm
    • 安装到项目根目录的node_modules
    • 使用require方法引入模块

流

  • 特点
    • 一种数据的传输手段
    • 区别于fs模块的直接读取
      • 分段读取，减少内存的占用
      • 可以控制读取速度
      • 可以边读取边写入
  • 操作
    • 可读流（createReadStream）
    • 可写流（createWriteStream）
    • 管道流（pipe）
