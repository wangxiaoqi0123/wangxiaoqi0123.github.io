websocket

  • 概念
    • H5新增的，基于TCP的一种新的网络协议
      • 使用HTTP协议（短连接）实现握手和分手
      • 使用一个长连接保持通信
    • 实现浏览器与服务器全双工通信
      • HTTP协议是单向的，靠轮询来保持客户端与服务端的同步
      • websocket是双向的，服务器可以主动向浏览器发送数据
    • 主要用来构建实时的web应用（聊天、弹幕、监控、推送等）
    • 使用方法简单，但低版本浏览器不兼容
  • 实现聊天室的功能
    • 服务端
      • 引入ws
      • new WebSocket.Server({port: 8888},cb)
      • ws.on(‘connection’,(client) => {})
      • client.on(‘close’,cb)
      • client.send(msg)
      • client.on(‘message’,cb)
      • client.close()
    • 客户端
      • new WebSocket(‘ws://localhost:8888’)
      • ws.onopen
      • ws.onclose
      • ws.onerror
      • ws.onmessage = (msg) => {}
      • 插入dom
      • 随机颜色

socket.io

  • 概念
    • 一个实现了实时通信的跨平台框架
    • 将websocket和轮询机制以及其它的实时通信方式封装成了通用的接口，websocket只是其中一个子集
    • 当websocket无法使用时，辅以ajax轮询等其他方式
    • 包括了服务器端的NodeJS和客户端的JavaScript
  • 使用
    • 服务端
      • 引入socket.io
      • 结合express
      • connection
      • socket
        • emit，broadcast
        • on
    • 客户端
      • 引入socket.io
      • connect
      • socket
        • emit
        • on
