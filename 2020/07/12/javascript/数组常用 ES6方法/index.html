数组常用 ES6方法

发表于 2020-07-12 更新于 2020-07-16 分类于 javascript

reduce(callback,initialValue)

  1. 作用 - 对数组每项进行迭代操作

  2. 参数说明

    参数 描述
    (total,currentValue,index,arr)=>{} 函数参数:
    参数 描述
    total 初始值, 或者计算结束后的返回值。
    currentValue 当前元素
    index 当前元素的索引
    arr 当前元素属于的数组对象
    initialValue 传递给函数的初始值

  3. 返回值 -数组中每一项迭代后的结果

  4. 影响 -不对原数组造成影响

map(callback)

  1. 作用 -对数组进行映射

  2. 参数说明

    参数 描述
    (currentValue, index,arr)=>{} 函数参数:
    参数 描述
    currentValue 当前元素的值
    index 当前元素的索引值
    arr 当前元素属于的数组对象

  3. 返回值 -返回计算完成后的新数组

  4. 影响 -不对原数组造成影响

filter(callback)

  1. -对数组进行筛选过滤

  2. 参数说明

    参数 描述
    (currentValue, index,arr)=>{} 函数参数:
    参数 描述
    currentValue 当前元素的值
    index 当前元素的索引值
    arr 当前元素属于的数组对象

  3. 返回值 -返回一个符合条件的新数组集合

  4. 影响 -不对原数组造成影响

find(callback)

  1. 作用 - 查找数组中的某一项

  2. 参数说明

    参数 描述
    (currentValue, index,arr)=>{} 函数参数:
    参数 描述
    currentValue 当前元素的值
    index 当前元素的索引值
    arr 当前元素属于的数组对象

  3. 返回值 -当数组中的元素在测试条件时返回 true 时, find() 返回符合条件的元素，之后的值不会再调用执行函数如果没有符合条件的元素返回 undefined

  4. 影响 -不对原数组造成影响
