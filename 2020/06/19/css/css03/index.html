阿里巴巴图标库

发表于 2020-06-19 更新于 2023-10-20 分类于 css

线上引用

  1. 查找图标添加至项目 进入项目

  2. 复制 Font class 在线生成 的地址
    //at.alicdn.com/t/font_1199553_u7298mmz27i.css

  3. 页面 link 引用 注意加 http:

  1. 标签 clas 添加 iconfont 和 对应图标名

  2. 当项目 添加新的 图标 在线地址要更新 页面的引用地址也要更新

下载引用

  1. 查找图标添加至项目 进入项目

  2. 点击下载 解压文件 选取 iconfont.css iconfont.eot iconfont.svg iconfont.ttf iconfont.woff 文件, 放在项目 assets 文件夹下 (最好新建一个文件夹 一起放进去)

  3. main.js 中引入 iconfont.css 文件

  4. 标签 clas 添加 iconfont 和 对应图标名

  5. 当项目 添加新的 图标 整个文件夹需要重新打包 替换

UI库扩展 iconfont

UI 一般也是icon一般也是使用字体图标 所以可以在可添加字体图标的组件上使用 icon=”iconfont icon-xxx” 方式扩展
