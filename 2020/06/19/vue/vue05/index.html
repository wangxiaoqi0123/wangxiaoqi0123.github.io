Vue 单文件组件

发表于 2020-06-19 更新于 2023-10-20 分类于 vue 2.0

  1. template 默认只能有一个根节点 可书写 指令 事件 差值表达式 filter 等等

  2. components 注册引入该模板文件 的局部组件

  3. props 接受父组件 传递的数据

  4. data 作为主要数据源

  5. computed 依赖其他数据源的值 计算出新值

  6. watch 监听数据发生变化

  7. created 组件 被实例化 完成

  8. created 页面节点渲染完毕

  9. methods 书写 function

  10. scoped 针对当前文件样式锁定
