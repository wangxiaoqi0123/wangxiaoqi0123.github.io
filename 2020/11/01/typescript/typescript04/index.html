  • 布尔类型（boolean）

  • 数组类型（array）
  • 元组类型（tuple）

boolean number string 简单类型的变量声明

  • 布尔类型 boolean
  • 数字类型 （整数、浮点数）number

数组和元组类型的变量声明

一 整个数组数据类型一致

  • :Array< number >
  • :string[]

二 整个数组数据类型不一致情况

  • 联合类型声明数组(number|string)[]

三 自由任意类型元素的数组

  • any[]

四 严格限定类型和长度的元组

  • :[string,number,boolean]

enum枚举类型

枚举用于表示固定的几个取值，如 性别 男女。

  • 定义运用
  • 默认值

用 | 指定多种类型

  • 当一个变量可能存在多个类型值时，可以使用 | 指定多个类型

any 任意类型

  • 当不清楚值得类型时，可以使用any，任何数据类型的值都可以赋值给any

null 和 undefined

  • undefined和null各自的类型分别为undefined和null

void 类型

  • void与any相反，表示没有任何类型，一般用于函数返回值。在TS中只有undefined可以赋值给void类型。

never 类型

  • 表示永远不存在的值类型，一般用于抛出异常或者根本不可能有返回值的函数。
